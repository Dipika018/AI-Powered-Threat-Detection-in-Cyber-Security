## ЁЯУ▒ Overview

**AI Threat Detection** is a real-time network intrusion detection system built on **Kali Linux**. It captures every live network packet using Scapy, extracts 49 network-level features, and classifies traffic into 9 named attack categories using a voting ensemble of XGBoost, LightGBM, and Random Forest тАФ all displayed on a live cyberpunk-styled web dashboard.

> **Built for B.Tech Cyber Security** тАФ tested against real nmap attacks on Metasploitable 2 in a VMware virtual machine lab.

| Attack | Description |
|---|---|
